# Security Policy

This repository provides example integrations and scaffold code.

It is not a security product, and Context Compiler is not represented here as a
complete security control.

## Reporting a Vulnerability

If you discover an unsafe example default, vulnerable sample configuration, or
other security-sensitive issue in this repository, please avoid posting full
exploit details publicly at first.

Open a GitHub security advisory for the repository when available.

If that is not available, open a regular issue with sensitive details removed
and request a private follow-up channel.

## Scope Notes

Useful reports usually include:

- the affected example or scaffold path
- the runtime or framework involved
- the unsafe default or exposed behavior
- the expected safer behavior
